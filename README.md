# Site Profissional - Antônio Trindade

Portfólio e perfil profissional de Antônio Trindade, especialista em automação de testes e qualidade de software.

## 🎯 Sobre o Projeto

Site pessoal desenvolvido para apresentar experiência profissional, áreas de atuação e facilitar contato com potenciais clientes e parceiros.

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização customizada com paleta escura/clara
- **JavaScript** - Interações e animações (jQuery, Vegas, Particleground)
- **Font Awesome** - Ícones de redes sociais
- **Roboto** - Tipografia principal

## 📂 Estrutura do Projeto

```
public_html/
├── index.html              # Página principal do portfólio
├── css/                    # Folhas de estilo
│   ├── base.css           # Reset e base styles
│   ├── main.css           # Estilos principais
│   ├── vendor.css         # Estilos de terceiros
│   └── fonts.css          # Webfonts
├── js/                     # Scripts
│   ├── main.js            # Lógica principal
│   ├── plugins.js         # Plugins (Vegas, Particleground, etc)
│   └── modernizr.js       # Feature detection
├── images/                 # Imagens e assets
├── treinoautomacao/       # Ambiente de prática para testes automatizados
└── inc/                    # Utilitários backend (não mais em uso)
```

## 🎨 Seções do Site

1. **Hero** - Apresentação e especialidades
2. **Quem sou eu** - Background profissional e competências
3. **Como atuo** - Abordagens e metodologias
4. **Experiências** - Histórico profissional resumido
5. **Contatos** - E-mail e localização
6. **Redes Sociais** - Links para perfis profissionais

## 🚀 Como Executar

1. Clone o repositório
2. Abra `public_html/index.html` em um navegador moderno
3. Ou sirva através de um servidor local (ex: `python -m http.server` ou Live Server no VS Code)

## 📝 Paleta de Cores

- **Fundo escuro**: `#0f1115` (corpo), `#0b0d11` (seções)
- **Fundo claro**: `#f4f6f8` (seções alternadas)
- **Destaque**: `#7bd8c6` (links e acentos)
- **Texto escuro**: `#1a1d23`, `#4a5568`
- **Texto claro**: `#e7e7ec`, `#c9d3de`

## 🎓 Área de Treino

O diretório `/treinoautomacao` contém páginas HTML interativas para prática de automação de testes, incluindo:
