# Proxima Implementacao - Menu Hamburger Retratil

Este ciclo define o refactor da navegacao do site para substituir o menu superior atual por um menu hamburger retratil, mantendo as mesmas opcoes e rotas existentes.

## Objetivo
- adotar um padrao de navegacao semelhante a apps como Gmail/GitHub.
- manter compatibilidade com o projeto estatico (HTML/CSS/JS vanilla).
- reduzir risco de regressao funcional e melhorar usabilidade mobile.

## Artefatos do ciclo
- plan.md: estrategia, escopo, arquitetura, riscos e criterios de aceite.
- tasks.json: backlog executavel com dependencias, ordem, riscos e validacoes.

## Referencias internas
- partials/main-menu.html
- js/core/menu-loader.js
- css/layout.css
- css/justified-nav.css
