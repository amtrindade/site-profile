# Canonical Cycle - Refactor de Menu

Esta pasta organiza o proximo ciclo de evolucao do projeto para eliminar duplicacao de menu entre paginas HTML.

## Artefatos

- `plan.md`: plano completo, pesquisa de tendencias, decisao tecnica e fases de execucao.
- `tasks.json`: backlog executavel do ciclo com dependencias e criterios de aceite.

## Objetivo do ciclo

Criar uma unica fonte de verdade para o menu, usando o menu da home (`index.html`) como padrao oficial, e reaproveitar esse mesmo menu em todas as paginas sem replicacao de markup.
